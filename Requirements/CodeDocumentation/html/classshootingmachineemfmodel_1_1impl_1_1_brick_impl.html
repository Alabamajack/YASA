Detailed Description

An implementation of the model object 'Brick'.

The following features are implemented:

  • Has Task Brick
  • Has Event Brick
  • Has Alarm Brick
  • Has HW Ports Brick
  • Has Inter Brick In Brick
  • Has Inter Brick Out Brick
